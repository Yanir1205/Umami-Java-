הוראות התקנה של הפרויקט:
1.	התקנת GIT (למקרה שלא מותקן כבר אצלכם במחשב):
a.	https://git-scm.com/book/en/v2/Getting-Started-Installing-Git
2.	הורדת הקבצים שנמצאים ב-GIT למחשב הלוקלי שלכם:
a.	פותחים תיקייה חדשה במחשב שבה רוצים שכל התוכן יהיה
b.	פותחים command line (CMD) בתיקייה שכרגע פתחנו ורושמים:
git init
git clone https://github.com/Yanir1205/Umami-Java-.git
3.	התקנת npm (למקרה שלא מותקן כבר אצלכם במחשב):
a.	יש לפתוח command line (CMD) בכל תיקייה ולרשום:
npm install -g npm
4.	התקנת כל התוספים של ה- frontend (node modules):
a.	יש לפתוח command line (CMD) בתיקייה של ה- frontend ולרשום:
npm i
5.	לאחר ההתקנה, יש לפתוח IDE שתומך ב- Java – לדוגמא IntelliJ ולפתוח את הפרויקט שנמצא בתוך תיקיית ה- java שהורדנו מ-GIT.
6.	יש להריץ את קוד ה- Java בעזרת ה- IDE
7.	יש לפתוח command line (CMD) בתיקייה של ה-frontend ולרשום:
npm start
הדפדפן שמוגדר אצלנו כברירת מחדל ייפתח באופן אוטומטי וניתן יהיה לראות את הפרויקט.
